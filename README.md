# Proyecto-FoodSave
#proyecto para la realizacion del trabajo final de Taller de Programacion Web
## 🛠️ Tecnologías Utilizadas
* **HTML**
* **CSS**
  * **Grid Layout** 
  * **Flexbox**
  * **Media Queries (`@media`)** 
  * **Animaciones (`@keyframes`)**
* **JavaScript**
* **Node.js**
* **Express.js**
* **SQLite**
* **Fetch API**
* **LocalStorage**
## Estructura del Proyecto
* `/css` - Hojas de estilo independientes y optimizadas sin caracteres especiales.
* `/img` - Recursos visuales y optimizados para los platos gastronómicos.
* `login.html` - Página para iniciar sesión.
* `registrar.html` - Página para crear una nueva cuenta.
* `index.html` - Página principal de bienvenida.
* `menu.html` - Carta digital con diseño adaptativo en rejilla.
* `ofertas.html` - Promociones especiales de la temporada.
* `form_pedido.html` - Formulario de pedidos validado.
* `pedidos.html` - Historial y comprobantes del usuario.
## Funcionalidades
* Registro de usuario.
* Inicio y cierre de sesión.
* Almacenamiento de usuario en SQLite.
* Registro de pedidos.
* Consulta del historial de pedidos.
* Visualización de contraseña.
* Diseño adaptable.
## Instalación
1. Clonar repositorio.
2. Instalar las dependencias: npm install
3. Iniciar servidor: node server/server.js
4. Abrir proyecto en el navegador.
